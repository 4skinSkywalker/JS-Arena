Write a query that generates a series of numbers from 1 to N, where N is the single value contained within the number table.

Structure

The information is stored in the following table:

  • number:
    • n: An integer value, which is the upper bound of the sequence.

Task

Write a query to generate a sequence of integers starting from 1 up to N, where N is the given value in the dataset. The query should leverage recursive SQL techniques to produce the desired outcome.

Example

Suppose the number table contains:

n
5

Output

The query should generate the following sequence:

n
1
2
3
4
5
