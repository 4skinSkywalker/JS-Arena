Julia conducted a 15 days of learning SQL contest.

Write a query to print the total number of unique hackers who made at least one submission each day (starting on the first day of the contest).

The query should print this information for each day of the contest, sorted by the date.

Input Format

The following tables hold contest data:

  • Hackers:
    • id – the id of the hacker
    • name – the name of the hacker
  • Submissions:
    • date – the date of the submission
    • id – the id of the submission
    • hacker_id – the id of the hacker who made the submission
    • score – the score of the submission

Tips

Search for "Gaps-and-Islands SQL" to find more information on how to solve this challenge.

Here is a schematic that illustrates how to identify islands:

Date       hacker_id   rn  island_key
--------------------------------------
2016-03-01         1    0  2016-03-01
2016-03-01         2    0  2016-03-01
2016-03-02         1    1  2016-03-01  <- Continuation (no gap)
2016-03-03         1    2  2016-03-01  <- Same key; continuation
--------------------------------------
2016-03-04         2    2  2016-03-02  <- New key; gap occurred

Explanation

On March 01, 2016 hackers made submissions. There are 4 unique hackers who made at least one submission each day.

On March 02, 2016 hackers made submissions. Now only two hackers submitted every day, so there are 2 unique hackers who made at least one submission each day.

On March 03, 2016 hackers made submissions. Again, only two hackers submitted every day.

On March 04, 2016 hackers made submissions. The same two hackers continued submitting daily.

On March 05, 2016 hackers made submissions. Only one hacker submitted every day, so there is 1 unique hacker.

On March 06, 2016 only one hacker made a submission. There is 1 unique hacker.
