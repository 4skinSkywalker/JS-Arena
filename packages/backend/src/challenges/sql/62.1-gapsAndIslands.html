Identify contiguous intervals of numbers (islands) and separations (gaps) within a given sequence from a database. The task is to return a list of intervals that represent islands, as well as concatenate the intervals representing gaps.

Structure

The sequence of numbers is stored in the sequence table:

  • value: an integer representing each element in the sequence.

Example

Here is an example of the sequence table:

value
1
2
3
7
8
9
10
20
21
30

Output

The query should return the following, specifying the type of each interval:

type start end
island 1 3
island 7 10
island 20 21
island 30 30
gap 3 7
gap 10 20
gap 21 30
