Your task is to merge overlapping or consecutive date intervals from a database. You'll be given a set of date ranges, and your goal is to combine them into the fewest possible intervals that represent the same overall coverage.

Structure

The date ranges are stored in the following table dates:

  • date_start: Represents the start date of an interval.
  • date_end: Represents the end date of an interval.

Example

Given the following dates table:

date_start date_end
'2020-01-01' '2020-01-31'
'2020-01-15' '2020-02-22'
'2020-03-01' '2020-03-31'
'2020-02-26' '2020-04-26'
'2020-04-27' '2020-04-30'
'2020-05-01' '2020-05-31'

Output

The query should return the following merged intervals:

start end
'2020-01-01' '2020-02-22'
'2020-02-26' '2020-04-29'
'2020-05-01' '2020-05-31'

For each data set, your query should effectively combine periods and produce results that correctly represent the continuous coverage of dates without unnecessary overlaps.
