Determine the average power usage for all sensors in a specified chain (e.g., Bedroom Lights -> Light Switch -> Breaker), the result should be rounded to the nearest integer. For better debugging, the path through each sensor will also be displayed.

Sort the results by the complete_path in lexicographic ascending order.

Structure

The information is stored in the following tables:

  • sensors: Contains the ID, name, and previous sensor ID in the chain.
  • events: Records the sensor ID and the power usage in Watts.

Example

Given the following sensors table:

id name prev_sensor_id
1 Breaker NULL
2 Mains 1
3 Bedroom Light Switch 1
4 Bedroom Lights 3
5 Bathroom Lights 1

And the following events table:

id sensor_id power_usage
1 2 1050
2 3 150
3 4 300
4 5 130

Output

The output should show each complete path and the average power usage rounded to the nearest integer:

complete_path avg_power_usage
Breaker <- Bathroom Lights 130
Breaker <- Bedroom Light Switch <- Bedroom Lights 225
Breaker <- Mains 1050
