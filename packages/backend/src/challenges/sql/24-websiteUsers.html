The users table contains the following columns:

  • id - The unique user's ID;
  • login - The unique user's login;
  • name - The user's name;
  • user_type - The user's role type (which can be "user", "admin", "moderator", etc.).

Your task is to return the number of users of each type.

Sort the results by the number of users of a given type in descending order, in case of a tie, sort by the type.

First Example

For the given table users

id login name user_type
1 Ann92 Ann user
2 Bob88 Bob user
3 Carol412 Carol admin
4 Deril39 Deril admin
5 EvaGrin Eva user

You should return the following table:

stats
There are 3 users
There are 2 admins

Second Example

For the given table users

id login name user_type
1 Ann92 Ann user
2 Bob88 Bob moderator
3 Carol412 Carol editor
4 Deril39 Deril admin
5 EvaGrin Eva admin

You should return the following table:

stats
There are 2 admins
There is 1 editor
There is 1 moderator
There is 1 user
