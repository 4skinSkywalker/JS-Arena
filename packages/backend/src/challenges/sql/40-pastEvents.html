You recently attended an event and noticed that your USB drive is missing. You are certain that you still had the drive before this most recent event, so you must have lost it at one of the events you attended earlier.

You remember having the USB drive within the last 7 days, so the event where you lost it must have occurred within the 7 days immediately preceding your most recent event.

The list of events you've attended (including the most recent one) is stored in a table called Events. It has three columns:

  • id: the unique id of the event;
  • name: the name of the event;
  • event_date: the date of the event.

Task

Return all events that meet the following conditions:

  • They occurred before the most recent event.
  • They took place within the 7-day period immediately preceding the most recent event date.
  • The most recent event itself must not be included.

The result should contain the columns name and event_date, and should be sorted by event_date in descending order.

Example

For the following table Events

id name event_date
1 TGIF '2016-11-18'
2 TGIF '2016-11-11'
3 Weekly team meeting '2016-11-07'
4 Weekly team meeting '2016-11-14'

the output should be

name event_date
Weekly team meeting '2016-11-14'
TGIF '2016-11-11'
